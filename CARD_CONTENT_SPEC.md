# 🎴 Card Content Specification

คู่มือกำหนดข้อมูลที่ต้องแสดงในการ์ดแต่ละประเภท

---

## 📋 การ์ดตัวละคร (Character Card)

### ข้อมูลที่ต้องมี (Required)

#### 1. Header Section (ส่วนบน)
```
┌─────────────────────────────┐
│ 🔥 #001        [⚖️ BAL]     │
│ Element  ID     Class Badge │
└─────────────────────────────┘
```
- **Element Icon** (มุมซ้าย): 🔥💧🌍💨⚡❄️
- **Card ID** (ซ้ายกลาง): #001, #002, etc.
- **Class Badge** (มุมขวา): 
  - 🛡️ TANK
  - ⚔️ DPS
  - 💚 SUP (Support)
  - ⚖️ BAL (Balanced)

#### 2. Character Art Section (ส่วนกลาง)
```
┌─────────────────────────────┐
│                             │
│          🐲                 │
│      (ขนาด 120px)           │
│                             │
└─────────────────────────────┘
```
- **Character Emoji/Art**: ใหญ่และชัดเจน
- **Background**: Gradient ตามธาตุ
- **Watermark**: Element symbol (โปร่งแสง)

#### 3. Name Section (ส่วนชื่อ)
```
┌─────────────────────────────┐
│    ไฟร์ดราก้อน              │ Thai Name
│    FIRE DRAGON              │ English Name
└─────────────────────────────┘
```
- **ชื่อภาษาไทย**: ฟอนต์ Kanit Bold 20pt
- **ชื่อภาษาอังกฤษ**: ฟอนต์ Outfit 12pt, uppercase

#### 4. Stats Section (ส่วนสถิติ)
```
┌─────────────────────────────┐
│ ❤️ HP: 450    ⚔️ ATK: 70   │
│ ⚡ SPD: 40    🛡️ Balanced   │
└─────────────────────────────┘
```
- **HP (Hit Points)**: พลังชีวิต
- **ATK (Attack)**: พลังโจมตีพื้นฐาน
- **SPD (Speed)**: ความเร็ว (กำหนดลำดับการโจมตี)
- **Class**: Tank/DPS/Support/Balanced

#### 5. Passive Skill Section (ส่วนสกิลติดตัว)
```
┌─────────────────────────────┐
│ 💫 PASSIVE SKILL            │
│ ผิวหนังไฟ                   │
│ ได้รับโล่ +40 ทุกรอบ        │
└─────────────────────────────┘
```
- **Badge**: "PASSIVE SKILL" หรือ "💫 พาสซีฟ"
- **Skill Name**: ชื่อสกิลภาษาไทย
- **Description**: คำอธิบายสั้นๆ (1-2 บรรทัด)

#### 6. Turn Order Indicator (ส่วนลำดับการเล่น)
```
┌─────────────────────────────┐
│         ┌───┐               │
│         │ 3 │               │
│         └───┘               │
│      Turn Order             │
└─────────────────────────────┘
```
- **Turn Order Number**: หมายเลขลำดับการโจมตี (1-6)
  - คำนวณจาก Speed stat
  - Speed สูงสุด = เล่นก่อน (เลข 1)
  - Speed ต่ำสุด = เล่นทีหลัง (เลข 6)
- **แสดงตำแหน่ง**: มุมบนขวาของการ์ด หรือ ด้านบนตัวละคร
- **สไตล์**: วงกลมหรือสี่เหลี่ยมพื้นสีตามธาตุ

#### 7. Element Advantage Section (ส่วนความได้เปรียบ)
```
┌─────────────────────────────┐
│ Strong vs: 💨 Wind (+20%)   │
│ Weak vs: 🌍 Earth (-20%)    │
└─────────────────────────────┘
```
- **Strong vs**: เอาชนะธาตุอะไร (ดาเมจ x1.2)
- **Weak vs**: แพ้ธาตุอะไร (ดาเมจ x0.8)

#### 8. Flavor Text Section (ส่วนข้อความตกแต่ง)
```
┌─────────────────────────────┐
│ "มังกรไฟผู้ยิ่งใหญ่         │
│  เอาชนะลมได้ด้วยเปลวเพลิง"  │
└─────────────────────────────┘
```
- **Quote/Lore**: คำพูดหรือเรื่องราวของตัวละคร
- **Style**: Italic, สีอ่อน

## 🃏 การ์ดสกิล (Skill Card)

### ข้อมูลที่ต้องมี (Required)

#### 1. Header Section (ส่วนบน)
```
┌─────────────────────────────┐
│ ⚡3  นรกเพลิง          🔥  │
│ Cost  Name          Element │
└─────────────────────────────┘
```
- **Energy Cost** (มุมซ้าย): ⚡0, ⚡1, ⚡2, ⚡3
  - ขนาดใหญ่ที่สุด (40x40px)
  - สีทอง พื้นดำ
- **Skill Name** (กลาง): ชื่อสกิลภาษาไทย
- **Element Icon** (มุมขวา): 🔥💧🌍💨⚡❄️

#### 2. Skill Art Section (ส่วนภาพสกิล)
```
┌─────────────────────────────┐
│                             │
│       🔥💥🔥                │
│    (Skill Icon)             │
│                             │
└─────────────────────────────┘
```
- **Skill Illustration**: รูปหรือไอคอนแสดงสกิล
- **Background**: เอฟเฟกต์ตามธาตุ
- **Size**: 140px height

#### 3. Stats Bar (ส่วนสถิติสกิล)
```
┌─────────────────────────────┐
│   ⚔️ 200      🛡️ 0         │
│   Damage      Shield        │
└─────────────────────────────┘
```
- **Damage (⚔️)**: ดาเมจที่สกิลทำได้
  - ถ้าเป็นลบ = Heal (💚)
- **Shield (🛡️)**: โล่ที่ได้รับ
- **ฟอนต์**: JetBrains Mono Bold 18pt

#### 4. Description Section (ส่วนคำอธิบาย)
```
┌─────────────────────────────┐
│ โจมตีศัตรูตัวที่มี HP       │
│ น้อยที่สุดด้วยเปลวไฟนรก     │
│ ที่ร้อนแรงที่สุด             │
│                             │
│ Target: Lowest HP           │
│ Effect: Burn (2 turns)      │
└─────────────────────────────┘
```
- **Description (Thai)**: คำอธิบายภาษาไทย 2-3 บรรทัด
- **Target Type**: ระบุเป้าหมาย
  - Default (Front)
  - Lowest HP
  - Highest HP
  - Back
  - Front
  - All (ทั้งหมด)
- **Special Effect**: เอฟเฟกต์พิเศษ (ถ้ามี)
  - Burn (ไฟลุก)
  - Stun (สตัน)
  - Slow (ช้าลง)

#### 5. Footer Section (ส่วนล่าง)
```
┌─────────────────────────────┐
│ 🐲 ไฟร์ดราก้อน             │
│ [🔥 ULTIMATE]              │
│ Character    Type Badge     │
└─────────────────────────────┘
```
- **Character Info** (ซ้าย):
  - Emoji + ชื่อตัวละครเจ้าของ
- **Type Badge** (ขวา):
  - 🔥 ULTIMATE (สีทอง)
  - ⚔️ NORMAL (สีน้ำเงิน)
  - 💫 PASSIVE (สีม่วง)

## 🎯 ระบบ Turn Order (ลำดับการเล่น)

### การคำนวณลำดับ

ลำดับการโจมตีในแต่ละรอบจะคำนวณจาก **Speed** ของตัวละคร

```
Speed สูงสุด → เล่นก่อน (Turn Order #1)
Speed ต่ำสุด → เล่นทีหลัง (Turn Order #6)
```

### ตัวอย่างการคำนวณ

สมมติมีตัวละคร 6 ตัวในสนาม:

| ตัวละคร | Speed | Turn Order |
|---------|-------|------------|
| ธันเดอร์เบิร์ด | 62 | 1️⃣ (เร็วสุด) |
| ไลท์นิ่งไทเกอร์ | 58 | 2️⃣ |
| วินด์อีเกิล | 60 | 3️⃣ |
| ไฟร์ดราก้อน | 40 | 4️⃣ |
| วอเตอร์ตอร์ทอยส์ | 35 | 5️⃣ |
| เอิร์ธไรโน | 30 | 6️⃣ (ช้าสุด) |

### การแสดงบนการ์ด

```
แบบที่ 1: วงกลมมุมบนขวา (แนะนำ)
┌─────────────────┐
│ 🔥      ┌───┐   │
│         │ 3 │   │
│         └───┘   │
└─────────────────┘

แบบที่ 2: Badge ด้านบน
┌─────────────────┐
│ 🔥  [TURN 3]    │
│                 │
└─────────────────┘

แบบที่ 3: แถบสี
┌─────────────────┐
│ 🔥 ▓▓▓ 3 ▓▓▓   │
│                 │
└─────────────────┘
```

### การแสดงในเกม (เหมือนรูปตัวอย่าง)

```
┌────────────────────────────────────────────────┐
│ Round 2                                        │
├────────────────────────────────────────────────┤
│  ┌──┐  ┌──┐  ┌──┐  ┌──┐  ┌──┐  ┌──┐         │
│  │🐷│  │🐷│  │🐰│  │🦎│  │🐢│  │🐻│         │
│  │ 1│  │ 2│  │ 3│  │ 4│  │ 5│  │ 6│         │
│  └──┘  └──┘  └──┘  └──┘  └──┘  └──┘         │
│  ตัวละครเรียงตามลำดับ Speed                   │
└────────────────────────────────────────────────┘
```

**คำอธิบาย:**
- แสดงตัวละครทั้งหมดในสนาม (ทั้ง Player และ Enemy)
- เรียงจากซ้ายไปขวา ตาม Speed (เร็วสุดอยู่ซ้ายสุด)
- แสดงหมายเลขลำดับใต้ตัวละคร
- ตัวที่กำลังเล่นจะมีเอฟเฟกต์พิเศษ (เรืองแสง/ขยาย)

### สีของ Turn Order Badge

```css
Turn 1-2 (เร็วมาก):
  - Background: #ffd700 (ทอง)
  - Text: #000000 (ดำ)
  - Border: 3px solid #ff8c00

Turn 3-4 (ปานกลาง):
  - Background: #4facfe (ฟ้า)
  - Text: #ffffff (ขาว)
  - Border: 2px solid #0080ff

Turn 5-6 (ช้า):
  - Background: #ff4d4d (แดง)
  - Text: #ffffff (ขาว)
  - Border: 2px solid #b91c1c
```

### กรณีพิเศษ

**Speed เท่ากัน:**
- ใช้ Attack เป็นตัวตัดสิน (Attack สูงกว่าเล่นก่อน)
- ถ้า Attack เท่ากันด้วย → สุ่ม

**Passive Skill ที่เพิ่ม Speed:**
- คำนวณ Turn Order หลังจากใช้ Passive แล้ว
- ตัวอย่าง: วินด์อีเกิล มี Passive +10 Speed
  - Base Speed: 60
  - Actual Speed: 70 (60 + 10)
  - Turn Order คำนวณจาก 70
